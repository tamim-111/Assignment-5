Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript technique where a single event listener is added to a parent element to handle events for its child elements, even if they are added later.

Question-4: What is event bubbling in the DOM?

Event bubble is a default js dom behavior. this behavior tell us when When an event occurs on an element, it first runs on that element and then moves up through its parent elements, all the way to the html document.
